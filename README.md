# System Description (300 w)
An automated tram system can boost the economy by reducing congestion, increase the value of surrounding land,
and encourage the development of new businesses. This supports UN's SDG 11 to make cities and human settlements
safe, inclusive, and sustainable.


- Selected SDG relevant problem
- Aim of the system and how it relates to the selected problem
- Required system functionalities, how they will solve the SDG

# System Design (2000 w)
The overall system, should include
•	The types of IoT nodes the system consists of, and their role in the system,
•	The network topology and computing paradigm to be used for the system.
•	Connectivity protocols used between devices and, if appropriate, the wider internet.
•	Messaging protocols and technologies to be used.
•	The control the use will have over the system setup/operation.
•	Any additional information of relevance to the system overall, e.g legal, privacy or other considerations
